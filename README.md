# Quill

Simple JSON logger backend for Elixir.

## Installation

The package can be installed by adding `quill` to your list of dependencies in
`mix.exs`:

```elixir
def deps do
  [
    {:quill, "~> 0.2.1"}
  ]
end
```

Additional docs can be found at
[https://hexdocs.pm/quill](https://hexdocs.pm/quill).
